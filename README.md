# 🧾 Code Audit - Aplikasi Kasir Sederhana

## 📌 Deskripsi
Project ini merupakan implementasi tugas **Code Audit (Clean Code)** dengan studi kasus aplikasi kasir sederhana berbasis Python (CLI).

Project ini terdiri dari:
- Kode awal (belum clean code)
- Kode hasil perbaikan (clean code)

---

## 📂 Struktur File
- `kasir_trash.py` → Kode awal (banyak pelanggaran clean code)
- `kasir_clean.py` → Kode hasil perbaikan sesuai prinsip clean code

---

## 🚀 Cara Menjalankan Program
1. Pastikan Python sudah terinstall
2. Jalankan perintah berikut di terminal:

```bash
python kasir_clean.py
